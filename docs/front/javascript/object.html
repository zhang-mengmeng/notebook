面向对象 ​

认识对象类型 ​

  • 对象类型是一种存储键值对（key-value）的复杂数据类型

  • 键值对可以是属性和方法，其中key是字符串，value可以是任何类型，包括基本数据类型，函数类型，对象类型等

创建对象的三种方式 ​

对象的遍历方式 ​

普通for循环遍历 ​

Object.keys()方法会返回一个由一个给定对象的自身可枚举属性组成的数组

js
var info = {
  name: "张萌",
  height: 1.88,
  age: 18,
  "my friend": "kobe",
  running: function() {
    console.log("running~")
  }
}

var infoKeys = Object.keys(info)
for (var i = 0; i < infoKeys.length; i++) {
  var key = infoKeys[i]
  var value = info[key]
  console.log(`key: ${key}, value: ${value}`)
}
